EventCast℠

This project is designed to help users plan their day by combining data from two APIs: OpenWeatherMap and SeatGeek. The program retrieves the user's location, checks for any events happening within a 20 mile radius, and checks the weather forecast for the day. The program will then display the event information and notify the user whether it is likely to rain during the event. This project aims to provide users with a simple and convenient tool for planning their day, taking into account both weather and local events.

For access to the github repo, please contact me directly as I do not desire to have the repo publically available with my API keys accessable.
